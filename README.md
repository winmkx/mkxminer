# mkxminer
Assembler-optimized Lyra2Z miner for AMD GPUs (ZCoin, GINcoin, Gentarium...)

!!! Only works with AMD videocards. No NVidia, Intel, CPU or ASICs !!!


=== Speeds* Lyra2z (no overclocking)
Vega64: 7.4MHs
RX 580: 3.7MHs
RX 480: 3.5MHs
R9 390: 1.8MHs
